# weather-dashboard

A centralized dashboard to keep track of multiple locations' weather conditions

Weather condition data and icons by [OpenWeather ](https://openweathermap.org/)

## Tracks:

- Current temperature
- Current humidity
- Current wind speed
- Current UV Index
- 5-Day forecast of temperature and humidity
